# LearnovateWeb
 Progetto web computing 2019-2020
 
 
 Learnovate è una web app utile per conoscere, insegnare e apprendere.
 Offre agli studenti corsi di formazione selezionati e svolti da professori e professionisti. 
 I corsi potranno essere seguiti direttamente online sul sito web ed essere salvati nella sezione “Miei corsi”, per essere usufruiti in qualsiasi momento senza ulteriori ricerche. 
 Il sito offre due livelli: Learnovate free e Learnovate premium. Come suggerisce il nome, con Learnovate Free, gli studenti potranno utilizzare il sito in modo gratuito, ma avranno accesso solo a determinati corsi, invece, con la modalità Premium, avranno a disposizione tutti i corsi presenti sul sito.
 Learnovate è stato sviluppato usando le tecnologie studiate durante il corso: HTML5, CSS3, JQuery, AJAX, JAVA Servlet, Bootstrap; sono solo alcune delle tante tecnologie utilizzate per lo sviluppo del sito web.
